# Siv3D-sample
## Overview
OpenSiv3Dで作成した、ちょっとしたものを置く場所です。  
アイデアが降ってきた/気が向いたときに更新していきます。  
## Requirement
OpenSiv3D (作成時の最新バージョンを使用)
## Usage
1. OpenSiv3Dをインストール
2. (Visual Studio使用時) [siv3d-sample.sln](./siv3d-sample.sln)を開く
3. 試したいサンプルの`Main.cpp`をプロジェクトに加えてビルド
